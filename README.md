# Transformer Warehouse Forecasting
This repository contains the technical implementation for my Master's dissertation at GISMA University of Applied Sciences.

## Project Overview
This research integrates predictive deep learning and Transformer architectures to solve supply chain volatility in 3PL warehouse operations.

## Contents
- `Final_Thesis_Pipeline_ChaithanyaVemuri.ipynb`: Complete PyTorch pipeline including data preprocessing, Positional Encoding, and the `ShiftTransformer` architecture.

- PyTorch
- Pandas & NumPy
