# Transformer Warehouse Forecasting
This repository contains the technical implementation for my Master's dissertation at GISMA University of Applied Sciences.

## Project Overview
This research integrates predictive deep learning and Transformer architectures to solve supply chain volatility in 3PL warehouse operations.

## Contents
- `Final_Thesis_Pipeline_ChaithanyaVemuri.ipynb`: Complete PyTorch pipeline including data preprocessing, Positional Encoding, and the `ShiftTransformer` architecture.

## Technical Stack
- PyTorch
- Pandas & NumPy
- Matplotlib & Seaborn

##How to Run the Pipeline
-Create a free account on Kaggle.

-Go to your Kaggle account settings and click "Create New API Token". This will download a personal kaggle.json file to your computer.

-Open the M5_Real_Transformer_Pipeline.ipynb notebook in Google Colab.

-Run the first cell. When the prompt appears, upload your personal kaggle.json file to securely authenticate the dataset download.
